# opal-drb

A dRuby implementation for Opal.

## Installation

    $ gem install opal-drb

## How to run example

    $ git clone https://github.com/youchan/opal-drb
    $ cd opal-drb/example
    $ bundle install
    $ bundle exec rackup

Open following url and watch development console.

    http://localhost:1234
    => ACK!
